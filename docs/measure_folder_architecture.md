# Power BI Measure Folder Architecture

To maintain semantic clarity and enterprise governance, measures are organised into structured folders:

## 1. Executive KPIs
- Total Revenue
- Total Operating Cost
- Route Profit
- Profit Margin %
- Total Passengers
- Revenue Forecast Accuracy %

## 2. Commercial KPIs
- Yield per Passenger
- Premium Cabin Revenue %
- Revenue per Flight
- Ancillary Revenue %
- Advance Booking Window
- Demand Momentum Index

## 3. Operational KPIs
- Load Factor %
- On-Time Departure %
- Cancellation Rate %

## 4. Financial KPIs
- Fuel Cost Ratio
- Cost per Route
- Route Risk Score

## 5. Governance KPIs
- Certified KPI Count
- % Certified KPIs

## 6. Forecast & Predictive KPIs
- Ticket Forecast Accuracy %
